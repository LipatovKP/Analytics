# 📊 Складской Аналитический Дашборд (Dash & Plotly)

Интерактивное веб-приложение для визуализации и анализа товарных запасов на основе данных из CSV-файла. Позволяет отслеживать цены, категории и общую стоимость склада в реальном времени.

## 🚀 Функционал
* **Интерактивные фильтры**: Выбор нескольких категорий и соответствующих им брендов.
* **Динамическая отчетность**: Автоматический расчет общей стоимости выбранных товаров (Цена × Остаток).
* **Визуализация**: 
    * Горизонтальный столбчатый график цен с цветовой кодировкой по категориям.
    * Круговая диаграмма распределения товаров (в разработке/опционально).
* **Подробные данные**: Просмотр артикулов, размеров и цветов при наведении на элементы графика.

## 🛠 Технологии
* **Python 3.x**
* **Pandas**: Обработка и фильтрация данных.
* **Plotly Express**: Создание интерактивных графиков.
* **Dash**: Веб-фреймворк для интерфейса.

# Создайте и активируйте виртуальное окружение
